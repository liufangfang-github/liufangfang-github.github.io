react-hook

useRef

useRef返回一个ref对象，对象只包含了一个current属性
useRef() 返回undefined，可以在创建的时候传入一个null/给一个初始值，useRef(null/初始值)，传入的参数就是对象中current的值。
useRef的变化不会主动渲染到页面

useImperativeHandle

useImperativeHandle在使用ref时 自定义暴露给父组件的实例值，使用子组件暴露的值/方法

useState

函数组件中定一个“state变量”，useState()返回当前定义的state和更新state的函数

useEffect

useEffect是异步的
一个组件中可以使用多个effect
componentDidMount() 方法会在组件已经被渲染到DOM中后运行，依赖列表为空时只会被调用一次，类似于vue中的mounted
**componentWillUnmount()**从DOM中移除时会调用，依赖列表为空时类似于vue中的destroy生命周期
componentDidUpdate() 会在更新后会被立即调用。首次渲染不会执行此方法。

useEffect(
  () => {
    // 这里的代码块 等价于 componentDidMount
    // do something...

    // return的写法 等价于 componentWillUnmount 当前组件被卸载时执行
    return () => {
       // do something...
    };
  },
  // 依赖列表，当依赖的值有变更时候，执行副作用函数，等价于 componentDidUpdate
  [ xxx，obj.xxx ]
);

依赖列表是灵活的，有三种写法

  • 当数组为空[]，表示不会因为页面的状态改变而执行回调方法，仅在初始化执行一次
  • 当这个参数不传递表示页面的任何状态一旦变更都会执行回调方法
  • 数组非空时，数组里的值一旦发生变化，就会执行回调方法

其他场景1:依赖某些值，但是不想在初始化时调用，只在依赖的值发生变化时调用，这里需要用到useRef 这个钩子

const firstLoad = useRef(true);
useEffect(() => {
  if (firstLoad.current) {
    firstLoad.current = false;
    return;
  }
  // do something...
}, [依赖项]);

useReducer

不是一个必须的hook，不用它也可以完成正常的需求开发，useReducer可以使我们的代码具有更好的可读性、可维护性

const [state, dispatch] = useReducer(reducer, initState);

useReducer接收两个参数：第一个参数是reducer函数，第二个参数是初始化的state

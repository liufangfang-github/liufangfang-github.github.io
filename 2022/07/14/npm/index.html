bin：命令名到本地文件名的映射 [https://www.freesion.com/article/45281433075/]

  • npm run 后发生了什么
    管理项目时会在package.json中，配置scripts属性。这里就是用来配置npm run XXX命令的，当执行npm run xxx时，会在scripts里找到xxx，如果是.js文件会直接执行。如果不是，我们在安装项目时会在.bin生成一个软连接，xxx时会在packge里找到对应的项目名称，然后在.bin里找到相应的xxxx.js文件，xxxx是一个软连接，链接到node_modules目录下找到xxxx包，找到包下的package.json文件里的bin

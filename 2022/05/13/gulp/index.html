gulp

Posted on 2022-05-13

gulp：基于流的打包构建工具；依赖node开发的。流： 流文件，一种文件的传输格式，如视频； 流格式

API

  • src 找到源文件
  • task 创建基于流的任务，
    gulp.task(任务名称，处理函数)
  • dest() 把内容放到指定目录内
  • watch():监听 并在发生更改时运行任务
  • series: series(…tasks) 逐个执行各个任务
  • parallel()， parallel(…tasks) 并行开始多个任务
  • pipe() 管道函数

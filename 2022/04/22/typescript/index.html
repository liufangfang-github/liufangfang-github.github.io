@Prop装饰器接受一个参数，这一个有三种写法。@Prop(options: (PropOptions | Constructor[] | Constructor) = {})

  • PropOptions ，包含以下选项：type(类型), required(必填), default(默认值), validator(验证函数)
  • Constructor[]， 指定prop的可选类型；
  • Constructor，如String，Number，Boolean，Array，Object，Date，Function，Symbol或自定义函数类型等，指定prop的类型
    如下面的例子
    
import { Component, Vue, Prop } from 'vue-property-decorator'
@Component
export default class ProductConfigItem extends Vue {
@Prop({ type:String, default: '1'}) tabType: String| undefined
@Prop(Number) readonly propA: number | undefined
@Prop({default(){return{}}, type: Object }) attrs: any

}

注：属性的类型后面需要加上undefined 类型；或者在属性名后面加上!，表示非null 和 非undefined的断言，否则编译器会给出错误提示；

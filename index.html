Posted on 2022-07-14

bin：命令名到本地文件名的映射 [https://www.freesion.com/article/45281433075/]

  • npm run 后发生了什么
    管理项目时会在package.json中，配置scripts属性。这里就是用来配置npm run XXX命令的，当执行npm run xxx时，会在scripts里找到xxx，如果是.js文件会直接执行。如果不是，我们在安装项目时会在.bin生成一个软连接，xxx时会在packge里找到对应的项目名称，然后在.bin里找到相应的xxxx.js文件，xxxx是一个软连接，链接到node_modules目录下找到xxxx包，找到包下的package.json文件里的bin

  • rangeBand -> bandwidth

d3是在svg中绘画， echarts是在Canvas中绘画
svg： 1. 矢量图，放大不会失真
2. 基于XML，可以为每个元素添加js事件处理器
3. 不适合游戏应用
Canvas： 1. 位图，放大会失真
2. 不支持事件处理器
3. 能够以png/jpg格式保存图像
4. 适合游戏应用

  • text-anchor 标签的对齐方式text-anchor=”start”时，(x,y)为的起始坐标。text-anchor=”middle”时，(x,y)为的中轴坐标。text-anchor=”end”时，(x,y)为的结束坐标。
    g标签是一个组合标签，将SVG形状组合在一起。

  • linearGradient用来定义线性渐变色，但是必须嵌套在defs的内部
  • d3.interpolateBlues(t)返回与blues的顺序配色方案相对应的RGB颜色字符串， t是介于0和1之间的数字。d3.interpolateBlues(0.4)
  • d3.interpolateRgb(a, b)用于对两种颜色进行插值，通过调整伽马值返回两种颜色之前的色值。d3.interpolateRgb(“Red”, “blue”)(0.2)

Posted on 2022-05-12

format输出文件格式

  • amd: 异步模块定义，用于像RequestJS这样的模块加载器。
  • cjs: CommonJS,适用于Node或Browserify/webpack。
  • es: 将软件包保存为ES模块文件。
  • iife: 一个自动执行的功能，适合作为script标签这样的，只能在浏览器中运行。
  • umd: 通用模块定义，以amd、cjs和iife为一体。
  • system: SystemJS加载器格式
  • 当format类型为iife或者umd格式的时候需要配置output.globals选项参数以提供全局变量名来替换外部导入*
